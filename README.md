# mes_orchidees
Projet fil rouge - Mes orchidées - Site web encyclopédique et gestion de collections.

Ce projet consiste en la création d'un site web d'une encyclopédie & gestion de collections.
Mes Orchidées n’est pas qu’un simple exercice de développement, c’est une réponse digitale concrète à une problématique de passionnés. 
Ce projet est né d'un constat simple : 
l'absence d'une plateforme centralisée capable de répondre aux exigences pointues de l'orchidophilie. 

L'objectif est de transformer la gestion complexe d'une collection botanique en une expérience fluide et intuitive. 
À travers cette application, je propose un outil d'excellence capable de centraliser des connaissances encyclopédiques précises tout en accompagnant l'utilisateur dans l'entretien quotidien de ses spécimens.
